---
title: <mimor>
---

Summoning the custom element `<mimor>` can render a mimor.

For example:

```xml
<mimor src="https://inner.xieyuheng.com/mimors/debug.mimor" />
```

Result:

<mimor src="https://inner.xieyuheng.com/mimors/debug.mimor" />

What is mimor?

It is a memory helping tool: <https://mimor.app>

- [poem-vertical] 齊物論 · 莊子

  - 大知閑閑 小知閒閒
    大言炎炎 小言詹詹
    其寐也魂交
    其覺也形開
    與接為構 日以心鬭

  - 縵者 窖者 密者
    小恐惴惴 大恐縵縵
    其發若機栝 其司是非之謂也
    其留如詛盟 其守勝之謂也

  - 其殺如秋冬 以言其日消也
    其溺之所為之 不可使復之也
    其厭也如緘 以言其老洫也
    近死之心 莫使復陽也
